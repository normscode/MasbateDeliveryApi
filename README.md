# MasbateDeliveryApi

An **ASP.NET Core Web API** for a delivery/service system (MasbateDelivery).  
Built as a learning/personal project to practice **C#/.NET Web API**, clean structure, and CRUD patterns.

---

## Tech Stack

- **.NET / ASP.NET Core Web API**
- **Entity Framework Core** (if your `Data/` folder contains DbContext + migrations)
- **SQL Server / LocalDB** (or whatever DB you configured)
- Swagger / OpenAPI (typical for ASP.NET Web API)
